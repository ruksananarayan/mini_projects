# Microclimate Crop Advisor

A real-time AI-powered agricultural advisory system that provides crop recommendations based on microclimate conditions.

## Features

- 🌍 Real-time location-based weather data
- 🤖 AI-powered crop predictions using machine learning
- 💬 Interactive chatbot for agricultural advice
- 📊 Comprehensive microclimate analysis
- 🎯 Crop suitability scoring
- 📋 Actionable recommendations

## Prerequisites

- Python 3.8+
- Node.js (for development)
- OpenWeatherMap API key (optional - mock data works without)
- OpenAI API key (optional - mock responses work without)
